Java Version: OpenJDK 25.0.1

How to Run the Program
To execute the application, navigate to the src directory of the project:
cd "..\Fair Billing\src"

Then run the program by supplying the log file path as a command-line argument:
java MainClass.java <path-to-log-file>

Replace <path-to-log-file> with the actual file location on your system.
